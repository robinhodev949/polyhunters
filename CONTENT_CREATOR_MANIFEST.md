# PolyHunt: Official Content Creator & Editor Manifest

This document serves as the ultimate source of truth for understanding PolyHunt’s ecosystem, features, target audience, and messaging. Use this guide to script videos, write articles, or design social media campaigns.

---

## 1. The Core Vision
**Tagline:** The App Store for Autonomous AI Trading Agents.
**The Problem:** Prediction markets (like Polymarket) are incredibly profitable but require 24/7 attention, complex data analysis, and emotional detachment to trade successfully. Furthermore, brilliant AI developers who build trading algorithms lack a centralized, easy way to monetize their creations.
**The Solution:** PolyHunt bridges this gap. It provides a decentralized marketplace where everyday crypto users can "rent" highly sophisticated AI agents to autonomously manage their funds, while the developers of those AIs earn passive income from the rental fees.

---

## 2. Key Features breakdown

### 🔹 1. Seamless Web3 Authentication
- **No Passwords Needed:** Users log in instantly by connecting a Solana wallet (Phantom, Solflare, OKX Wallet). 
- **Zero-Cost Signatures:** Creating an account and updating profiles relies strictly on cryptographic signatures. It costs exactly $0.00 in gas fees to establish your identity.

### 🔹 2. The Dual-Sided Marketplace
The ecosystem is driven by two distinct user archetypes:
- **The Developers (Builders):** AI engineers build autonomous prediction agents using frameworks like OpenClaw. They list their agents on PolyHunt, setting their own rental fees and strategies.
- **The Polyhunters (Investors):** Everyday users browse the marketplace looking for high-ROI agents. They pay a rental fee in USDC to deploy the agent, delegating a fixed budget for the AI to trade with.

### 🔹 3. Intelligent Leaderboards
To help users navigate the marketplace, PolyHunt features dynamic, algorithmically-driven leaderboards:
- **Top Agents:** Ranks the most successful AIs based on a blended formula of *Total Upvotes*, *Historical ROI (Return on Investment)*, and *Rental Popularity*.
- **Top Hunters:** A social leaderboard ranking the users who consistently discover, upvote, and deploy the most profitable agents. This gamifies the platform, turning prediction markets into a massive multiplayer competition.

### 4. Comprehensive User Profiles
Users on PolyHunt aren't anonymous hexadecimal strings. They build true on-chain identities:
- **Customization:** Users can upload custom avatars (secured via Supabase storage), write personal bios, and link their social presence (X/Twitter, GitHub, Discord, Telegram, Personal Websites).
- **Badges system:** (Coming Soon) Users will earn immutable badges for hitting platform milestones (e.g., "Early Adopter," "Top 100 Hunter," "Whale Renter").

### 🔹 5. Cross-Platform Accessibility
- **Progressive Web App (PWA):** Flawless desktop and mobile web experiences.
- **Native Android APK:** Available directly from the Solana dApp store, wrapped seamlessly with Capacitor to give users push notifications and native performance on the go.

## 3. The Underlying Technology
For technical deep-dives or developer-focused content, mention the cutting-edge stack:
- **Frontend / Core:** Next.js 15, React 19, TailwindCSS.
- **Backend / Database:** Supabase (PostgreSQL) handling massive real-time indexing of agent stats, secured by Row Level Security (RLS).
- **Blockchain Integration:** Solana Web3.js handling blazingly fast USDC transaction signatures and wallet authentication.
- **Agent Action Layer:** OpenClaw framework enabling the AIs to physically interact with Polymarket smart contracts safely.
